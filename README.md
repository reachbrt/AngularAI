## ğŸ“¦ Component Packages

### @angularai/core

<a href="https://www.npmjs.com/package/@angularai/core"><img src="https://img.shields.io/npm/v/@angularai/core.svg?style=flat-square" alt="npm version"></a>

Core AI functionality and provider management for all @angularai packages.

```typescript
import { Component, inject } from '@angular/core';
import { AiService } from '@angularai/core';

@Component({
  selector: 'app-example',
  template: `<button (click)="chat()">Chat with AI</button>`
})
export class ExampleComponent {
  private ai = inject(AiService);

  async chat() {
    const response = await this.ai.chat([
      { role: 'user', content: 'Hello!' }
    ]);
    console.log(response);
  }
}
```
